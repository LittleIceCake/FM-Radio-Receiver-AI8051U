# 串口通讯协议说明

## 📌 概述

AI8051U 硬件串口通讯协议说明。通过串口发送命令可控制 FM 收音机模块的各项功能。

## 🔧 硬件参数

- **串口配置**：115200-8-N-1（波特率115200，8位数据，无校验，1位停止位）
- **物理接口**：UART（具体引脚参见硬件原理图）

## 📋 命令列表

| 命令            | 功能           | 示例                            | 说明                       |
| --------------- | -------------- | ------------------------------- | -------------------------- |
| `Freq_set:`     | 设置频率       | `Freq_set:108.5`                | 设置频率（单位：MHz）      |
| `Freq_get`      | 获取频率       | `Freq_get`                      | 返回当前频率               |
| `Band_switch:`  | 频带切换       | `Band_switch:1`                 | 切换频带（参数含义见备注） |
| `Freq_add:`     | 频率增加       | `Freq_add:0.1`                  | 增加指定步长（单位：MHz）  |
| `Freq_sub:`     | 频率减少       | `Freq_sub:0.1`                  | 减少指定步长（单位：MHz）  |
| `Time_set:`     | 时间设定       | `Time_set:15:30`                | 设置时间（24小时制）       |
| `Date_set:`     | 日期设定       | `Date_set:2026-03-03`           | 设置日期                   |
| `Datetime_set:` | 日期与时间设定 | `Datetime_set:2026-03-03 15:30` | 同时设置日期和时间         |
| `Help`          | 帮助           | `Help`                          | 显示所有可用命令           |

> **注**：所有命令均以换行符（`\n` 或 `\r\n`）结尾。
